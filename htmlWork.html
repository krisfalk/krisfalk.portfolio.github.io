    - I can understand the usefulness of a batch file, but may never use again. -


    --> Batch file 1. Creates a new local user account and then deletes it.
    --> Batch File Content:
    - @ECHO off
    - title Creating new local user account.
    - set /p _user=Please type the name of the new user and press 'Enter':
    - net user %_user% /add
    - pause >nul
    - echo change password
    - net user %_user% *
    - echo press 'Enter' to delete.
    - pause >nul
    - net user %_user% /del


    --> Batch file 2. Merges two .txt documents into one .txt document.
    --> Batch File Content:
    - @echo off
    - set _firstText=%1
    - set _secondText=%2
    - set /p _thirdText=Type what merged text document should be called and press 'Enter':
    - copy %_firstText%+%_secondText% %_thirdText%
